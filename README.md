# 🩺 Diabetes Prediction System

A machine learning–powered web application that predicts whether a person is at risk of diabetes based on key medical parameters. The application features an intuitive interface built with Streamlit, allowing users to make predictions for individual patients or analyze multiple records at once.

---

## 📌 Project Overview

This application is designed to assist users in estimating diabetes risk using commonly collected health indicators. By entering patient information or uploading a dataset, users can receive quick predictions generated by a trained machine learning model.

### Key Features

### 🔹 Individual Prediction

Enter patient details such as glucose level, BMI, age, blood pressure, and other medical attributes. The model analyzes the inputs and predicts whether the patient is likely to have diabetes, along with the confidence score.

### 🔹 Bulk Prediction

Upload a CSV file containing multiple patient records. The application processes the entire dataset, predicts the diabetes status for each record, and generates a downloadable file with the prediction results.

### 🔹 Model Information

The app also includes an overview of the machine learning model, dataset, and the significance of each medical feature used during prediction.

---

## 📊 Dataset

The model is trained using the *PIMA Indians Diabetes Dataset, a widely used healthcare dataset containing medical information collected from female patients. The dataset includes **768 patient records* with several diagnostic measurements that help determine diabetes risk.

---

## 🤖 Machine Learning Model

The prediction system is built using the *Logistic Regression* algorithm, a reliable supervised learning technique for binary classification problems.

### Model Training

* Training Data: 80% of the dataset
* Testing Data: 20% of the dataset
* Average Accuracy: *Approximately 78–80%*

Although the model provides reliable predictions, it is intended for educational and research purposes only and should not replace professional medical diagnosis.
